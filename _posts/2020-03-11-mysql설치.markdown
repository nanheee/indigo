### 설치 가능 MySQL 버전 확인
```
$ brew search mysql
```

### MySQL 설치
```
$ brew install mysql
```

최신 버전이 아닌 이전 버전으로 설치하고 싶으면 아래와 같이 입력하면 된다.
```
$ brew install mysql@5.7
```

### MySQL 설정
```
$ mysql.server start
```

```
$ mysql_secure_installation
```

### MySQL 사용
```
$ mysql -u root -p
```
